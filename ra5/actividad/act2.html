Ejercicio: Creación de un Repositorio y Scripts de Copia de Seguridad

Objetivo:

  • Practicar la creación de una cuenta en GitHub.
  • Crear un repositorio en GitHub.
  • Desarrollar scripts de copia de seguridad para sistemas operativos Windows y Linux.
  • Probar los scripts y subirlos al repositorio de GitHub.

Instrucciones:

  1. Crear una cuenta en GitHub:

    Si aún no tienes una cuenta en GitHub, regístrate en github.com utilizando tu dirección de correo electrónico.

  2. Crear un Repositorio:
    • Después de iniciar sesión en GitHub, haz clic en el botón "New" en la esquina superior derecha de la página.
    • Asigna un nombre al repositorio, por ejemplo, "script".
    • Selecciona la opción para inicializar el repositorio con un archivo README.
  3. Desarrollar Scripts de Copia de Seguridad:
    • Crea dos scripts: uno para Windows y otro para Linux.
    • Utiliza ChatGPT como herramienta para generar los scripts. Por ejemplo, pide a ChatGPT que genere un script de copia de seguridad para Windows y otro para Linux.
    • Los scripts deben realizar una copia de seguridad de un archivo específico o de un directorio a otro lugar en el sistema.
    • Los scripts deben tener nombres descriptivos como "backup_windows.bat" y "backup_linux.sh".
  4. Probar los Scripts:
    • Ejecuta los scripts en sus respectivos sistemas operativos para asegurarte de que funcionen correctamente.
    • Verifica que se haya realizado la copia de seguridad correctamente.
  5. Subir los Scripts al Repositorio:
    • Agrega los scripts al repositorio creado en GitHub.
    • Realiza un commit con un mensaje descriptivo que indique lo que se ha añadido o modificado.
    • Haz un push de los cambios al repositorio remoto en GitHub.

Entrega:

Una vez completados los pasos anteriores, comparte el enlace de tu repositorio de GitHub en el formulario para que pueda revisar tus scripts y la configuración del repositorio.
