Actividad: Creación de un sistema dual boot con Windows 10 y Linux

Objetivo:

Crear un sistema dual boot con Windows 10 y Linux en una computadora, utilizando herramientas de particionado y copias de seguridad.

Entrega:

Un documento que incluya los pasos seguidos para crear el sistema dual boot, junto con capturas de pantalla de cada paso importante.

Pasos a seguir:

  1. Preparación:
    • Descargar Ventoy y las imágenes ISO de Windows 10 y una distribución de Linux (como Mint, Ubuntu, Linux Lite o Puppy Linux), así como la herramienta de Rescuezilla para usar GParted.
    • Preparar una unidad USB grande para crear una unidad booteable utilizando Ventoy.
  2. Creación de la unidad USB booteable:
    • Instalar Ventoy en la unidad USB y copiar las imágenes ISO descargadas a la unidad.
  3. Configuración del disco duro:
    • Arrancar desde la unidad USB booteable y seleccionar GParted desde el menú de Ventoy.
    • Crear una tabla de particiones GPT en el disco duro.
    • Redimensionar las particiones existentes para crear una partición de 50 GB para Windows y una partición de 30 GB para Linux, además de una partición de 2 GB para la memoria de intercambio (swap) de Linux.
  4. Instalación de Windows 10:
    • Reiniciar la computadora y arrancar desde la unidad USB booteable de Windows 10.
    • Instalar Windows 10 en la partición NTFS creada de 50 GB, siguiendo las instrucciones en pantalla.
  5. Instalación de Linux:
    • Reiniciar la computadora y arrancar desde la unidad USB booteable de la distribución de Linux.
    • Instalar Linux en la partición ext4 creada de 30 GB, asignando 2 GB para la partición de intercambio (swap) durante el proceso de particionado.
    • Configurar particiones adicionales según sea necesario, como /home, /boot o cualquier otra partición específica de Linux.
  6. Configuración del gestor de arranque:
    • Configurar GRUB o el gestor de arranque de tu distribución de Linux para detectar automáticamente la instalación de Windows y permitirte seleccionar entre ambos sistemas operativos al arrancar.
  7. Configuración de UEFI y Secure Boot:
    • Si tu ordenador utiliza UEFI en lugar del tradicional BIOS, ajusta la configuración del firmware para deshabilitar Secure Boot y configurar el modo de arranque adecuado para que tanto Windows como Linux se instalen y arranquen correctamente.
  8. Consideraciones de compatibilidad de hardware:
    • Verifica la compatibilidad de hardware de tu ordenador con las distribuciones de Windows y Linux que planeas instalar, asegurándote de que los controladores necesarios estén disponibles y de que no haya problemas conocidos con tu hardware específico.
  9. Configuración de red y otros ajustes de sistema:
    • Después de la instalación, configura la red, ajusta la configuración de energía, instala controladores adicionales y realiza otros ajustes según tus preferencias y necesidades específicas.
  10. Entrega:
    • Crea un documento que detalle los pasos seguidos para crear el sistema dual boot.
    • Incluye capturas de pantalla de cada paso importante, como la creación de la tabla de particiones GPT, el particionado del disco, la instalación de Windows 10 y Linux, la configuración del gestor de arranque y cualquier ajuste adicional realizado.
